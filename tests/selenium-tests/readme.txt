Here's a quick start to selenium web unit tests


    * Copy tests/etc/selenium to the web server that your application is on.

    * Copy tests/selenium-tests/tests to the web server that your application is on.



	* Go view /JSPWiki/selenium/TestRunner.html
	
	* Select the testsuite  ../tests/JSPWikiTestSuite.html and press go.
	
Done.
